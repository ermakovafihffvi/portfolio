Это приложение vue для интернет магазина одежды. Дизайн был предоставлен на курсе по продвинутому js на факультете веб-разработки платформы geekbrains. 

Чтобы запустить корректно: 
- нужно открыть через ide (VS, например),
- запусить сервер в "командной строке разработкчика" (важно при этом находится в папке проекта) с помощью команды nodemon server/server.js,
- убедиться что IIS Express остановлен,
- запусить проект через IDE (нажать зелёную треугольную кнопочку),
- убедиться что в браузере в url будет написано localhost:3000 (нужен именно этот порт), поправить вручную, если будет указано другое.

Что работает: 
- Корзина. Можно добавлять, удалять товары, считается общая стоимость, можно очистить всю корзину. Работает как иконка, так и сама страница корзины;
- Реализован вывод каталога из json файла c помощью ассинхронного запроса;
- Работают выпадающие списки.
