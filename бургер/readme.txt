Это просто одна страничка для заказа бургера. Можно выбрать приправы и стоимость заказа автоматически посчитается. Проект реализован на языке javascript версии es6. 

This is a single page for ordering a burger. You can choose seasonings and sauce then total price will be automatically counted. This project is released in js language of the es6 version.
